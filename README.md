# Frontend Developer Quiz — Nikhil (Upgraded Version)

This is an upgraded, clean and simple frontend prototype of a quiz system created by Nikhil for a 15-day internship submission.

## Features
- 5 Pages (Home, Quiz, Result, Data, About)
- Clean blue & white design with dark mode
- Client-side scoring stored in sessionStorage
- No animations; simple and professional UI
- Easy to extend and deploy (static site)

## How to run
1. Extract the ZIP.
2. Open `index.html` in a modern browser.
3. Use the theme toggle in the header to switch between light/dark.
4. Start the quiz and submit to see results.
